HTML 5

HTML 5 was first released over ten years ago, it has taken time for browsers and the people who use them to catch up with it, now it is reinventing personal computing. With HTML 5 not only can new tags be created but it also has it's own storage, caching and database and offers tools such as web workers so that we can access these new features.
